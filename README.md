# cn-qiumi-guide

本仓库用于归档和发布多个独立 HTML 页面，主要目的是整理和展示与特定主题相关的静态内容。

## 项目简介

`cn-qiumi-guide` 是一个静态页面归档仓库，存放若干独立制作的 HTML 文件。这些页面不依赖特定后端服务，可直接在浏览器中打开或通过静态托管服务访问。

仓库内容以信息整理与展示为主，不涉及商业推广、用户数据收集或交互式服务。

## 目录说明

```
cn-qiumi-guide/
├── index.html          # 仓库入口页面（如有）
├── pages/              # 各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源文件（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/` 目录下存放所有独立页面，每个页面为一个完整的 HTML 文档。
- `assets/` 目录存放页面共用的资源文件，按类型组织。

## 页面归档说明

- 每个页面均为独立 HTML 文件，可单独访问。
- 页面内容基于公开可获取的信息整理而成，仅供学习与参考。
- 页面之间无强关联，可独立使用或修改。
- 所有页面均不包含自动跳转、弹窗或第三方跟踪代码。
